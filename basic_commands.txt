git init - Initialize an empty git repository
git add 'file 1' - add 'file 1' to the new git repository
git commit -m 'message' - add a commit message
git log - show a history of the commit 
git status - show the current state of the repository

git remote - shows the avaliable remote location in a repository
git remote add 'remote_name' 'remote_url' - adds a new remote repository
git push 'remote_name' 'branch_name' - sends your code to the repository and branch specified
git pull 'remote_name' 'branch_name' - downloads code from then repositoryand branch specified
git clone 'repository_name' - downloads code from the repository

git branch - shows all the branches in a repository
git branch 'branch_name' - switches the current branch to the specified branch
git checkout 'branch_name' -  switches the current branch to the specified branch
